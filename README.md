# Proyecto Sprint 7
En este proyecto realizado por Daniela Izaguirre se utilizaron diferentes herramientas para crear entornos de desarrollo como:
*Anaconda
*VS Code
*Jupyter Notebooks.
Se realizó el análisis del dataset __vehicles_us__ en el cual venían datos sobre diferentes tipos de vehículos y sus características físicas. Para este análisis se utilizaron las librerías:
*pandas
*numpy
*plotly express
Posteriormente se realiza la programación de una aplicación web en *Streamlit* para mostrar el análisis de dichos datos con dos botones; un botón para mostrar un **histograma** donde se aprecia la cantidad de vehículos y su kilometraje y otro para un **gráfico de dispersión** que muestra el tipo de vehículo y su precio.
